Smart Search
============

Type an exact ProDy function name (for example ``parsePDB``).
If an exact match exists, you will be redirected directly to that function.
Otherwise, normal search results will be shown.

.. raw:: html

   <input id="smart-search-input"
          placeholder="Enter function name (e.g. parsePDB)"
          style="width:100%; padding:10px; font-size:16px;">

   <button onclick="smartSearch()" style="margin-top:10px;">
     Search
   </button>

   <script src="_static/smart_search.js"></script>
